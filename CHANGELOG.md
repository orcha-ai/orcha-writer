# Changelog

Orcha Writer 的版本变更记录。

## [Unreleased]

## [0.1.3] - 2026-05-10

### 新增

- 新增自动更新发布配置，生成并上传 `latest.json` 与 updater 签名产物。
- 新增系统菜单「调试模式」，正式包中也可打开 WebView DevTools。
- 新增工作区侧栏宽度拖拽，并记住上次调整后的宽度。

### 改进

- 模型供应商支持 OpenAI-compatible、Anthropic、Gemini、Ollama 和 Custom 类型。
- 模型请求地址改为按用户输入原样请求，不再自动拼接路径。
- 应用版本号改为以 `package.json` 为单一来源，同步到 Tauri 与 Cargo 配置。
- 更新检查显示运行时真实版本，避免打包后仍显示旧版本。
- 顶部标签栏会自动滚动到当前激活文件。
- 设置页高级操作文案改为「清理草稿缓存」和「重置基础设置」。
- 发布说明不再显示未配置 Apple Developer ID / Windows 代码签名提示。

### 修复

- 修复 OpenAI-compatible 请求地址重复拼接 `/chat/completions` 导致 404 的问题。
- 修复 release workflow 中 updater JSON 与资源命名参数未生效的问题。
- 修复工作区侧栏拖拽宽度没有实际生效的问题。

## [0.1.2] - 2026-05-10

### 新增

- 新增 AI 写作面板，支持当前文档上下文、选中文本上下文和手动输入。
- 新增智能体与快捷指令体系，支持 `/` 命令、选区浮层和结果卡片操作。
- 新增 OpenAI-compatible 模型调用通道，支持模型供应商、模型配置和 API Key 凭据配置。
- 新增深度思考开关、思考预算设置和思考过程展示。
- 新增 AI 模块 README 截图与产品说明。

### 改进

- 优化 AI 结果卡片操作区，支持应用修改、插入、新建文档、复制和重新生成。
- 优化对话自动滚动，避免生成长内容时打断用户手动滚动。
- 优化工作区、大纲和 AI 面板的收起交互，并记住工作区与大纲显示状态。
- 优化工作区启动逻辑，默认恢复上次打开的目录。
- 优化草稿保存闭环，草稿会优先保存到当前工作区并刷新文件树。
- 优化 README 内容结构，补充界面预览和 AI 写作说明。

### 修复

- 修复设置页输入框无法正常粘贴的问题。
- 修复 AI 回复重复显示的问题。
- 修复文本选中后没有背景色的问题。
- 修复执行选区快捷指令后选区状态丢失的问题。
- 修复工作区和大纲隐藏按钮状态不持久的问题。

## [0.1.1] - 2026-05-10

### 新增

- 新增命令面板与常用编辑、视图、插入命令入口。
- 新增代码块主题配置，并补充代码高亮样式。
- 新增应用内更新检查与安装流程的基础支持。

### 改进

- 优化搜索与替换面板的交互和视觉状态。
- 优化顶部工具栏的导出、插入和视图切换操作。
- 优化预览、导出、本地图片资源路径和展示体验。
- 底部状态栏现在会实时显示当前光标行列与文档字数。

### 修复

- 修复 macOS 打开事件处理的边界判断。
- 修复本地图片在预览与导出中的兼容问题。

## [0.1.0] - 2026-05-09

### 新增

- 首个公开构建版本。
- 支持 Markdown 编辑、预览、文件管理和跨平台安装包构建。
