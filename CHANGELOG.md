# Changelog

Orcha Writer 的版本变更记录。

## [Unreleased]

## [0.1.12] - 2026-05-15

### 新增

- AI 对话输入框新增发送前上下文预览，支持自动、选区和全文范围切换。

### 优化

- 关闭未保存文档或草稿时增加确认保护，覆盖标签关闭、右键批量关闭和菜单关闭。
- AI 对话结果应用后会高亮编辑器变更范围，并减少选区 AI 浮窗遮挡正文的情况。
- AI 对话流式输出改为阶段性内存更新，减少生成过程中频繁写入配置文件。
- 优化 AI 对话清空、失败重试和结果 Diff 交互，支持清空确认、编辑后重试、模型设置入口和行级差异展示。
- 调整 AI 对话上下文展示层级，发送前显示“将引用”，发送后仅保留轻量的“上次引用”回看。

## [0.1.11] - 2026-05-14

### 新增

- 外观设置新增“大纲位置”，支持右侧、左侧标签和关闭；左侧标签模式下，大纲显示在最近文件右侧。

### 优化

- 工作区目录和已打开文件会定时从磁盘刷新，外部修改后能更及时反映到界面。
- 顶部标签页右键菜单新增关闭当前标签和关闭所有标签。
- 关于页面版本号独立成行展示，并补充“版本号”标签。
- AI 对话的 Clear Chat 操作改为删除图标。
- 优化块编辑模式输入、表格编辑、多选和块移动时的渲染性能。

## [0.1.10] - 2026-05-13

### 新增

- 左侧工作区和最近文件支持图片、PDF 预览标签页。
- 图片预览支持缩放、适应窗口、实际大小、复制路径和系统打开。
- PDF 预览支持内嵌查看、复制路径和系统打开。

### 优化

- 代码块预览新增格式化操作，并调整到复制按钮左侧。
- 多语言文案补齐到工作区、编辑器、预览、AI 与设置页等主要界面。
- 优化右键菜单拦截策略，保留对应功能区的自定义右键菜单。
- 修复最近文件点击后不能切换到对应标签页的问题。

## [0.1.9] - 2026-05-13

### 优化

- 移除文档自动保存功能，保留手动保存和未保存状态提示。
- 退出应用前如果存在未保存文档或草稿，会弹出确认提示，避免误退出丢失内容。

## [0.1.8] - 2026-05-13

### 优化

- 优化块编辑模式首次打开后的块高度计算，减少块底部多余留白。
- 优化块编辑模式右键交互，避免原生菜单与块级菜单冲突。
- 优化块编辑模式拖拽排序体验，改为更稳定的指针拖拽，并增强插入位置提示线。

## [0.1.7] - 2026-05-12

### 新增

- 块编辑模式新增表格网格视图，Markdown 表格会展示为可编辑的表格块。
- 表格块支持直接编辑表头与单元格，并支持增删行列。
- 块编辑模式搜索新增关键词高亮，普通块与表格单元格都会标出命中词。
- 块级 AI 入口改为弹出对话框，支持快捷指令与自定义要求。

### 优化

- 优化块编辑模式下列表解析、列表标记展示和嵌套列表保真。
- 优化块编辑模式输入性能，减少编辑时的重复解析与隐藏预览渲染。
- 优化块编辑模式搜索体验，改进命中块定位、搜索替换、焦点保持和大跨度滚动定位。
- 优化 AI 对话中的润色结果展示，原文与修改后内容改为上下排列。
- 优化源码模式选区 AI 对话输入框，与块编辑 AI 对话保持一致的多行输入体验。

## [0.1.6] - 2026-05-12

### 新增

- 新增 Notion 式块编辑模式，支持 Markdown 分块编辑、斜杠菜单、块工具栏、拖拽排序和块级右键菜单。
- 新增块多选能力，支持范围选择、多块复制、删除、移动，以及复制为 Markdown / 纯文本。
- 新增块级 AI 指令入口，支持润色、扩写、缩短、转列表和生成下一块。
- 新增全局语言设置，默认使用中文界面与中文右键菜单。

### 改进

- 代码块预览新增复制操作、语言选择与深色模式适配。
- 块编辑模式接入 Markdown AST 辅助分块，增强 Frontmatter、HTML、数学公式、表格、Setext 标题和多行列表项保真。
- 优化图片块展示，块编辑模式下直接显示图片预览、路径输入和描述输入。
- 优化 AI 设置交互，模型配置改为先选大模型厂商再选模型，添加智能体和添加模型改为抽屉打开。
- 优化文件与工作区设置中的默认工作区输入框高度。
- 优化命令面板输入框高度与输入体验。

### 修复

- 修复全选会选中整个页面的问题，现在只作用于 Markdown 编辑区域。
- 修复块编辑模式下图片只显示文件名或描述的问题。
- 修复右键菜单点击其他区域不能关闭的问题。

## [0.1.5] - 2026-05-10

### 新增

- 新增 AI 对话流式输出，正文与深度思考过程会随服务端响应实时更新。
- 新增生成中的「取消」操作，可中断当前 OpenAI-compatible / Custom 流式请求。
- 关于页面新增当前应用版本号展示。

### 改进

- OpenAI-compatible 请求会在用户填写的地址后统一补齐 `/chat/completions`，并避免重复拼接。
- 模型供应商与模型配置的「测试」按钮改为真实请求当前模型服务。
- AI 服务错误信息补充实际请求地址，便于排查代理路径与服务商地址问题。

### 修复

- 修复深度思考过程没有流式展示的问题。
- 修复发送中无法取消生成、取消后仍可能继续写入回复的问题。

## [0.1.4] - 2026-05-10

### 修复

- 修复 Windows 发布任务中 Tauri CLI 调用失败的问题，重新发布完整安装包。
- 本版本包含 0.1.3 的全部功能与修复；0.1.3 的 GitHub Release 产物不完整，请以 0.1.4 为准。

## [0.1.3] - 2026-05-10

### 新增

- 新增自动更新发布配置，生成并上传 `latest.json` 与 updater 签名产物。
- 新增系统菜单「调试模式」，正式包中也可打开 WebView DevTools。
- 新增工作区侧栏宽度拖拽，并记住上次调整后的宽度。

### 改进

- 模型供应商支持 OpenAI-compatible、Anthropic、Gemini、Ollama 和 Custom 类型。
- 模型请求地址改为按用户输入原样请求，不再自动拼接路径。
- 应用版本号改为以 `package.json` 为单一来源，同步到 Tauri 与 Cargo 配置。
- 更新检查显示运行时真实版本，避免打包后仍显示旧版本。
- 顶部标签栏会自动滚动到当前激活文件。
- 设置页高级操作文案改为「清理草稿缓存」和「重置基础设置」。
- 发布说明不再显示未配置 Apple Developer ID / Windows 代码签名提示。

### 修复

- 修复 OpenAI-compatible 请求地址重复拼接 `/chat/completions` 导致 404 的问题。
- 修复 release workflow 中 updater JSON 与资源命名参数未生效的问题。
- 修复工作区侧栏拖拽宽度没有实际生效的问题。

## [0.1.2] - 2026-05-10

### 新增

- 新增 AI 写作面板，支持当前文档上下文、选中文本上下文和手动输入。
- 新增智能体与快捷指令体系，支持 `/` 命令、选区浮层和结果卡片操作。
- 新增 OpenAI-compatible 模型调用通道，支持模型供应商、模型配置和 API Key 凭据配置。
- 新增深度思考开关、思考预算设置和思考过程展示。
- 新增 AI 模块 README 截图与产品说明。

### 改进

- 优化 AI 结果卡片操作区，支持应用修改、插入、新建文档、复制和重新生成。
- 优化对话自动滚动，避免生成长内容时打断用户手动滚动。
- 优化工作区、大纲和 AI 面板的收起交互，并记住工作区与大纲显示状态。
- 优化工作区启动逻辑，默认恢复上次打开的目录。
- 优化草稿保存闭环，草稿会优先保存到当前工作区并刷新文件树。
- 优化 README 内容结构，补充界面预览和 AI 写作说明。

### 修复

- 修复设置页输入框无法正常粘贴的问题。
- 修复 AI 回复重复显示的问题。
- 修复文本选中后没有背景色的问题。
- 修复执行选区快捷指令后选区状态丢失的问题。
- 修复工作区和大纲隐藏按钮状态不持久的问题。
